
# Projeto: Desenvolvimento de História em Quadrinhos com IA

##Objetivo
Desenvolver uma história em quadrinhos utilizando ferramentas de Inteligência Artificial, aplicando metodologias ágeis (SCRUM e Kanban) e utilizando o GitHub para organização do projeto.

---

##Sobre a História
Título: Sombras do Algoritmo

Sinopse:
Um investigador experiente começa a suspeitar que uma inteligência artificial está manipulando provas criminais. Ao seguir pistas digitais, ele descobre que o verdadeiro inimigo não é humano — e pode estar em todo lugar.

---

## Ferramentas Utilizadas

- ChatGPT → criação de roteiro e personagens  
- IA de geração de imagens → criação das cenas da HQ  
- GitHub → organização e versionamento  
- Canva → montagem da HQ  

---

## Metodologia Utilizada

### SCRUM
O projeto foi dividido em Sprints, permitindo o desenvolvimento incremental da HQ.

Sprints utilizadas:
- Sprint 1: Definição da história e personagens  
- Sprint 2: Criação do roteiro  
- Sprint 3: Geração de imagens com IA  
- Sprint 4: Montagem da HQ  
- Sprint 5: Documentação do projeto  

---

### Kanban
Foi utilizado um quadro Kanban no GitHub para organização das tarefas, com as seguintes colunas:

- Backlog  
- A Fazer  
- Em Progresso  
- Concluído  

As tarefas foram organizadas em Issues e distribuídas nas Sprints.

## Considerações Finais

O uso de Inteligência Artificial permitiu acelerar o processo criativo, enquanto as metodologias ágeis ajudaram na organização e divisão das etapas do projeto.

O GitHub foi essencial para o controle das tarefas e acompanhamento do progresso.
